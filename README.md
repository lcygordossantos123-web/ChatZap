# ChatZap

## Para configurar o Firebase

1. Vá para https://console.firebase.google.com
2. Crie um novo projeto chamado **ChatZap**
3. Ative **Autenticação por Telefone**
4. Crie um Firestore Database em modo de teste
5. Copie o config do Firebase e coloque no App.js
